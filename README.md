# GeoJson PHP Library

> This package is based on Jeremy Mikola's [jmikola/geojson](https://github.com/jmikola/geojson/) package.

This library implements the
[GeoJSON format specification](https://geojson.org/).

## Installation

The library is published as a
[package](https://packagist.org/packages/eriksulymosi/geojson) and is installable via
[Composer](https://getcomposer.org/):

```
$ composer require "eriksulymosi/geojson"
```
